Movie Search Engine

Веб-приложение для поиска фильмов с различными фильтрами

Основные возможности:

  • Поиск по частичному совпадению названия
  • Фильтрация по году или диапазону лет
  • Исключение контента для взрослых
  • Выбор из 16 жанров
  • Топ популярных запросов
  • Адаптивный веб-интерфейс

Стэк

  • Python 3.12.4
  • Модуль http.server
  • Модуль logging
  • Модуль Jinja2
  • MySQL
  • Ванильный JavaScript

Установка

git clone https://github.com/SomeCrab/imdb_web
pip install -r requirements.txt
# Set up database configuration in .env file
python run.py
